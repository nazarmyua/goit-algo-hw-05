
# Порівняння алгоритмів пошуку рядків

## Опис роботи

У цій роботі проведено експериментальне порівняння трьох класичних алгоритмів пошуку підрядка в тексті:

- **Boyer–Moore**
- **Knuth–Morris–Pratt (KMP)**
- **Rabin–Karp**

Метою є оцінка їхньої продуктивності на реальних текстах українською мовою для випадків:
- коли шуканий шаблон **існує** в тексті;
- коли шаблон **відсутній** у тексті.

Тестування виконано на двох текстах (`article1`, `article2`) з різними за довжиною та структурою шаблонами.

---

## Методологія

Для кожної пари *текст – шаблон* алгоритм виконувався багаторазово. Для аналізу використовувалися такі метрики:

- **Avg Time** – середній час виконання одного запуску;
- **Total Time** – сумарний час виконання всіх запусків.

Час вимірювався у секундах.

## Аналіз результатів

1. **Boyer–Moore** стабільно демонструє найкращу продуктивність як для існуючих, так і для неіснуючих шаблонів.
2. **KMP** працює повільніше, але гарантує лінійну складність.
3. **Rabin–Karp** є найповільнішим через обчислення хешів.
4. Для неіснуючих шаблонів час виконання суттєво зростає.

---

## Висновки

- **Boyer–Moore** є найефективнішим для практичного використання.
- **KMP** підходить для стабільного пошуку з передбачуваною складністю.
- **Rabin–Karp** доцільний при пошуку багатьох шаблонів.
